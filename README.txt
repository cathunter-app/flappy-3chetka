Flappy Cat: Istanbul Run
==========================

How to play
-----------
1. Open `index.html` in a desktop or mobile browser.
2. Press **Start**, then tap/click/press **Space** to flap.
3. Fly through the gaps between buildings to score.
4. Works offline; sounds are generated by the browser (no audio files).

Notes
-----
- Mobile browsers often require a first tap to unlock audio. Your first tap does that.
- The cat sprite uses your provided image (`assets/cat.png`).
- Everything is canvas-based and scales to screen size.

Tuning knobs (open game.js):
- baseW: cat size (default 88*DPR)
- baseGap: starting gap (default max(260*DPR, 24% screen height))
- flap/gravity constants control feel
